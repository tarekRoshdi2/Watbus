# تحديثات النظام (System Updates) - الإصدار الأخير 🚀

## 📋 نظرة عامة على التحديثات
تم في هذا التحديث إطلاق دعم المراسلة عبر تليجرام، وتطوير ذكاء وكيل المبيعات (AI Swarm)، بالإضافة إلى إصلاح مشاكل المزامنة مع قاعدة البيانات ومشاكل الاتصال بمخدمات الذكاء الاصطناعي.

## ✨ الميزات الجديدة (New Features)
- **دعم تليجرام بوت (Telegram Bot Engine):**
  - إضافة دعم كامل للرد التلقائي على رسائل تليجرام بربط مباشر (Polling).
  - يمكن الآن إعداد التوكن الخاص بـ `@BotFather` من لوحة تحكم المنصة وتفعيله فوراً.
  - إضافة مسار API جديد `api/telegram/test-bot/` لاختبار اتصال البوت بنجاح.

## 🛠️ التحسينات (Enhancements)
- **تطوير ذكاء الذكاء الاصطناعي (ChatCore Swarm):**
  - الاستغناء عن الردود الثابتة المزعجة عند كتابة عبارات قصيرة مثل (ألو، تمام، شكراً). تم ربط هذه العبارات بـ Gemini لفهم السياق والرد بذكاء واختصار.
  - إضافة ميزة **الوعي بالسياق المتقدم (Context Awareness)** لمنع إعادة سرد باقات المبيعات الطويلة إذا كان العميل قد ناقشها بالفعل.

## 🐛 إصلاح الأخطاء (Bug Fixes)
- **إصلاح مشكلة انقطاع اتصال Gemini (Fetch Failed):**
  - تم إجبار نظام DNS في `Node.js` على استخدام `IPv4` كأولوية قصوى (`dns.setDefaultResultOrder('ipv4first')`)، مما حل مشكلة انقطاع الاتصال المفاجئ (ECONNRESET) عند محاولة السيرفر الاتصال بمخدمات Google.
- **إصلاح خطأ Prisma Foreign Key Constraint (P2003):**
  - تم معالجة انهيار مزامنة الأجهزة عند ربط أجهزة تخص مستخدمين افتراضيين (مثل `admin-tarek` و `meta-ai`). النظام الآن يتجاهل معرّف المستخدم الافتراضي لضمان حفظ الأجهزة بنجاح في `PostgreSQL` دون التسبب في خطأ بجدول `users`.
