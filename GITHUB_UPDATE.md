# تحديثات النظام (System Updates) - الإصدار الأخير 🚀

## 📋 نظرة عامة على التحديثات الأخيرة (2026-07-24)
تم في هذا التحديث إنجاز حزمة من الإصلاحات الهامة واستكمال تجهيز ملفات الرفع الخاصة بالحسابات والأجهزة:

### 🐛 1. إصلاح عدم ظهور رسائل الراسل (العميل) في صندوق المحادثات:
- **المشكلة:** كانت الرسائل الواردة عبر Meta Cloud API / Webhooks تُجيب عليها البوتات تلقائياً، ولكن لا يتم حفظ الرسائل الواردة أو بثها لشاشة الواجهة.
- **الحل:** تم تعديل `globalIncomingHandler` في `server.ts` لضمان حفظ الرسائل الواردة فوراً في قاعدة البيانات وبثها عبر `WebSocket` تزامناً مع استلامها.

### 🤖 2. منع تكرار الرسائل الترحيبية:
- **المشكلة:** كان البوت الذكي يعيد ديباجة الترحيب في كل رسالة جديدة.
- **الحل:** تم ربط تاريخ المحادثات بالكامل والتحقق من وجود ردود سابقة، مع تطبيق قواعد `STRICT ANTI-REPETITION` في الـ System Prompt لمنع البوت من الترحيب المكرر في المحادثات القائمة.

### 👥 3. تخصيص وإبراز صفحة الموظفين وفريق العمل:
- **التحسين:** إعادة هيكلة القائمة الجانبية في `App.tsx` وتسميات `translations.ts` لفصل:
  - 👥 **الموظفين (فريق العمل)**: إدارة الحسابات والموظفين البشر والصلاحيات (`ClientsPage`).
  - 🤖 **الموظفين (الذكاء الاصطناعي)**: مركز تحكم البوتات الذكية (`AgentsDashboard`).
